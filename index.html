SAP B1 Query Library

Tested SQL queries — copy, paste, run directly in SAP B1 Query Manager.

FREESales

Sales Order Register

All sales orders with customer, item, quantity, price and status. Date range filter included.

-- Sales Order Register By Digital Prime SELECT T0."DocEntry", T0."DocNum" AS "SO No.", T0."DocDate" AS "SO Date", T0."DocDueDate" AS "Delivery", T1."CardName" AS "Customer", T2."ItemCode", T2."Dscription" AS "Item", T2."Quantity", T2."Price", T2."LineTotal" AS "Amount", CASE WHEN ROW_NUMBER() OVER (PARTITION BY T0."DocEntry" ORDER BY T2."LineNum") = 1 THEN T0."DocTotal" ELSE NULL END AS "DocTotal", T0."DocStatus" FROM ORDR T0 INNER JOIN OCRD T1 ON T0."CardCode" = T1."CardCode" INNER JOIN RDR1 T2 ON T0."DocEntry" = T2."DocEntry" WHERE T0."DocDate" BETWEEN [%0] AND [%1] ORDER BY T0."DocDate" DESC, T0."DocEntry", T2."LineNum"

How to use in SAP B1:

  1. Open SAP B1 → Tools → Queries → Query Manager
  2. Click New → paste the SQL code above
  3. When prompted, enter From Date and To Date
  4. Click Run (F5) to execute
  5. Export to Excel using the table icon if needed

💡 Works on SAP B1 9.x and 10.0 · SQL Server only

FREEFinance

Accounts Receivable Ageing

Customer-wise outstanding in 0–30, 31–60, 61–90 and 90+ day buckets.

-- AR Ageing SELECT T0.[CardCode], T0.[CardName], SUM(CASE WHEN DATEDIFF(day,T0.[DocDueDate],GETDATE()) BETWEEN 0 AND 30 THEN T0.[DocTotal]-T0.[PaidToDate] ELSE 0 END) AS '0-30', SUM(CASE WHEN DATEDIFF(day,T0.[DocDueDate],GETDATE()) BETWEEN 31 AND 60 THEN T0.[DocTotal]-T0.[PaidToDate] ELSE 0 END) AS '31-60', SUM(CASE WHEN DATEDIFF(day,T0.[DocDueDate],GETDATE()) > 60 THEN T0.[DocTotal]-T0.[PaidToDate] ELSE 0 END) AS '60+', SUM(T0.[DocTotal]-T0.[PaidToDate]) AS 'Total Outstanding' FROM OINV T0 WHERE T0.[DocStatus] = 'O' GROUP BY T0.[CardCode], T0.[CardName] ORDER BY SUM(T0.[DocTotal]-T0.[PaidToDate]) DESC
FREEInventory

Stock Summary by Warehouse

On-hand, committed, on-order and available stock across all SAP B1 warehouses.

-- Stock Summary SELECT T0.[ItemCode], T0.[ItemName], T1.[WhsCode] AS 'Warehouse', T1.[OnHand] AS 'In Stock', T1.[IsCommited] AS 'Committed', T1.[OnOrder] AS 'On Order', (T1.[OnHand]-T1.[IsCommited]) AS 'Available' FROM OITM T0 INNER JOIN OITW T1 ON T0.ItemCode = T1.ItemCode WHERE T1.[OnHand] > 0 AND T0.[InvntItem] = 'Y' ORDER BY T0.[ItemCode], T1.[WhsCode]
FREEPurchase

Purchase Invoice Register with GST

Vendor-wise purchase invoices with item details, GST amount and vendor invoice number.

-- Purchase Invoice Register SELECT T0.[DocNum], T0.[DocDate], T1.[CardName] AS 'Vendor', T0.[NumAtCard] AS 'Vendor Invoice', T2.[ItemCode], T2.[Dscription], T2.[Quantity], T2.[Price], T2.[LineTotal] AS 'Taxable', T0.[VatSum] AS 'GST', T0.[DocTotal] FROM OPCH T0 INNER JOIN OCRD T1 ON T0.CardCode = T1.CardCode INNER JOIN PCH1 T2 ON T0.DocEntry = T2.DocEntry WHERE T0.[DocDate] BETWEEN [%0] AND [%1]
