Free & Premium SAP B1 SQL Queries

SAP B1 Query Library

Ready-to-use, tested SQL queries for SAP Business One. Copy, paste, run.

FREESales

Sales Order Register

All sales orders with customer, item, quantity, price, and status. Supports date range via SAP B1 Query Manager parameters.

-- Sales Order Register (Date Range) SELECT T0.[DocNum] AS 'SO Number', T0.[DocDate] AS 'Date', T0.[DocDueDate] AS 'Delivery Date', T1.[CardName] AS 'Customer', T2.[ItemCode], T2.[Dscription] AS 'Item', T2.[Quantity], T2.[Price], T2.[LineTotal] AS 'Amount', T0.[DocTotal], T0.[DocStatus] FROM ORDR T0 INNER JOIN OCRD T1 ON T0.CardCode = T1.CardCode INNER JOIN RDR1 T2 ON T0.DocEntry = T2.DocEntry WHERE T0.[DocDate] BETWEEN [%0] AND [%1] ORDER BY T0.[DocDate] DESC
FREEFinance

Accounts Receivable Ageing

Customer-wise outstanding in 0–30, 31–60, 61–90 and 90+ day buckets based on invoice due date.

-- AR Ageing Report SELECT T0.[CardCode], T0.[CardName], SUM(CASE WHEN DATEDIFF(day,T0.[DocDueDate],GETDATE()) BETWEEN 0 AND 30 THEN T0.[DocTotal]-T0.[PaidToDate] ELSE 0 END) AS '0-30 Days', SUM(CASE WHEN DATEDIFF(day,T0.[DocDueDate],GETDATE()) BETWEEN 31 AND 60 THEN T0.[DocTotal]-T0.[PaidToDate] ELSE 0 END) AS '31-60 Days', SUM(CASE WHEN DATEDIFF(day,T0.[DocDueDate],GETDATE()) > 60 THEN T0.[DocTotal]-T0.[PaidToDate] ELSE 0 END) AS '60+ Days', SUM(T0.[DocTotal]-T0.[PaidToDate]) AS 'Total Outstanding' FROM OINV T0 WHERE T0.[DocStatus] = 'O' GROUP BY T0.[CardCode], T0.[CardName] ORDER BY SUM(T0.[DocTotal]-T0.[PaidToDate]) DESC
FREEInventory

Stock Summary by Warehouse

Item-wise on-hand, committed, on-order, and available stock across all SAP B1 warehouses.

-- Stock Summary by Warehouse SELECT T0.[ItemCode], T0.[ItemName], T1.[WhsCode] AS 'Warehouse', T1.[OnHand] AS 'In Stock', T1.[IsCommited] AS 'Committed', T1.[OnOrder] AS 'On Order', (T1.[OnHand]-T1.[IsCommited]) AS 'Available' FROM OITM T0 INNER JOIN OITW T1 ON T0.ItemCode = T1.ItemCode WHERE T1.[OnHand] > 0 AND T0.[InvntItem] = 'Y' ORDER BY T0.[ItemCode], T1.[WhsCode]
FREEPurchase

Purchase Invoice Register with GST

Vendor-wise purchase invoices with item details, taxable amount, GST total and vendor invoice number.

-- Purchase Invoice Register with GST SELECT T0.[DocNum] AS 'PINV No', T0.[DocDate], T1.[CardName] AS 'Vendor', T0.[NumAtCard] AS 'Vendor Invoice No', T2.[ItemCode], T2.[Dscription], T2.[Quantity], T2.[Price], T2.[LineTotal] AS 'Taxable Amt', T0.[VatSum] AS 'GST Amount', T0.[DocTotal] AS 'Total' FROM OPCH T0 INNER JOIN OCRD T1 ON T0.CardCode = T1.CardCode INNER JOIN PCH1 T2 ON T0.DocEntry = T2.DocEntry WHERE T0.[DocDate] BETWEEN [%0] AND [%1] ORDER BY T0.[DocDate]
